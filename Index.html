¿Quieres saber por qué?

Esta página está creada para guiar y ayudar a los alumnos que quieran estudiar, practicar y mantener un orden mientras lo hacen. Las materias que aqui aparecen son escogidas de forma subjetiva y se pueden cambiar con unos mínimos conocimientos de html y css. A la derecha se muestran barras de progreso de las materias que se hayan incluido a mano. Con el mismo sistema funcionan los porcentajes.

Esta idea surge a raiz de las complicaciones que observó el autor de esta página, un estudiante de 2º de Bachillerato, que le costaba mantener un orden basandose en los metodos de estudios convencionales. El sistema educativo actual implica que los exámenes se realicen con muy poco tiempo de separación entre las pruebas de evaluación de las diferentes asignaturas.

Pese a su voluntad de poder organizarse teniendo en cuenta el poco margen de estudio, consideró necesario una herramienta para la búsqueda de respuestas concisas y concretas para un mayor rendimiento del potencial del estudiante. Esperamos que el usuario obtenga el beneficio para el cual fue creado y que le sirva de ayuda.
